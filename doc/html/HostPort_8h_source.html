2/*
3Class for managing hostPC-MCU transfer operations. To initialize the object call e.g.
4HostPort myhost(&Serial, 0xFFFFFFFF, 0xFFFFFFEE)
5where Serial is the used port for communication (USB in this case), 0xFFFFFFFF are 4 bytes (uint32) of start bytes (i.e. header),
6while 0xFFFFFFEE are 4 bytes (uint32) of stop bytes (i.e. terminator).
7To attach an object to trasmit use in the setup
8myhost.attachTx((uint8_t*) object, sizeof(object))
9where (uint8_t*) object is the object pointer (i.e. the RAM address of its first byte), while sizeof(object) its size (in bytes).
10The number of objects that can be trasmitted is MAX_NUM (defined in this file), while the max number of bytes is BUF_SIZE (defined
11in this file), including header and terminator bytes.
12Use myhost.write() to transmit the object (usually used in the loop)
13Simiarly, an object can be attached and receive
14myhost.attachRx((uint8_t*) object, sizeof(object)) //in the setup
15myhost.read() //in the loop
16
17Class written by SL, 11-22-2021
18*/
